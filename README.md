# Weather

Developed using HTML, JavaScript, jQuery, AJAX, Bootstrap & CSS.

### Description

This web application retrieves and displays the current weather conditions for the following cities:

 - Pretoria
 - Johannesburg
 - Cape Town
 - Soweto
 - Durban
